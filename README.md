Traditional voting systems face challenges such as vote manipulation, lack of transparency, centralized control, and identity fraud.  
This project addresses these issues by leveraging **Blockchain technology** and **Smart Contracts** to store votes immutably and **Face Verification** to authenticate voters before casting a vote.
